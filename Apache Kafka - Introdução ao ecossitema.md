# Apache Kafka é...

 ...uma plataforma de _streaming_ de eventos distribuídos de código aberto usada por milhares de empresas para:
 - _pipelines_ de dados de alto desempenho
 - análise de _streaming_
 - integração de dados
 - aplicativos de missão crítica
